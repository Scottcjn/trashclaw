# Windows Support for TrashClaw

This document provides instructions for setting up and running TrashClaw on Windows environments.

## Prerequisites

1. Python 3.7 or higher
2. Install pyreadline3 for enhanced command line interface on Windows:

   ```bash
   pip install pyreadline3
   ```

## Running on Windows

- TrashClaw should run seamlessly on Windows with the necessary fallbacks provided for Windows-specific modules.
- Ensure that you have `pyreadline3` installed to improve terminal interaction on Windows. If `pyreadline3` is unavailable, TrashClaw will still function but without some enhanced features.
