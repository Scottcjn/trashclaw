# Contributing to trashclaw

Thank you for your interest in contributing to trashclaw!

## Getting Started

### Prerequisites
- Python 3.8+
- Git

### Setup
1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/[YOUR_USERNAME]/trashclaw.git
   cd trashclaw
   ```
3. Create a new branch:
   ```bash
   git checkout -b your-feature-branch
   ```

## Making Changes
- Follow PEP 8 style
- Run `python trashclaw.py --help` to test
- Add tests if applicable

## Submitting a Pull Request
1. Push your branch to your fork
2. Open a PR against `Scottcjn/trashclaw`
3. Reference the bounty issue #1605

## Code of Conduct
Be respectful and follow standard open source practices.
