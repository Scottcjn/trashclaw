# Contributing to TrashClaw

Thank you for your interest in contributing to TrashClaw!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/trashclaw.git`
3. Create a branch: `git checkout -b feature/your-feature-name`

## Setup

`bash
pip install -r requirements.txt
`

## Making Changes

- Write clear, readable code
- Add comments where necessary
- Follow existing code style

## Submitting a PR

1. Push your branch: `git push origin feature/your-feature-name`
2. Open a Pull Request against `main`
3. Describe what your PR does and why

## Code Style

- Use Python PEP8 conventions
- Keep functions small and focused
- Write descriptive variable names

## Reporting Issues

Open a GitHub Issue with:
- A clear title
- Steps to reproduce
- Expected vs actual behavior

## Questions?

Open an issue or join the discussion on the bounty board.
